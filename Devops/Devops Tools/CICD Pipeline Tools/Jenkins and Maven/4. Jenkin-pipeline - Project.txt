## 3ï¸âƒ£ Create a Jenkinsfile in GitHub Repo

Example `Jenkinsfile`:

```groovy
pipeline {
    agent any   // run on any available Jenkins agent/node
    stages {
        stage('Clone Repo') {
            steps {
                git branch: 'main', url: 'https://github.com/username/myrepo.git'
            }
        }
        stage('Build') {
            steps {
                echo "Building the project..."
                sh 'mvn clean install'  // example for Java project
            }
        }
        stage('Test') {
            steps {
                echo "Running tests..."
                sh 'mvn test'
            }
        }
        stage('Deploy') {
            steps {
                echo "Deploying application..."
                // deploy script/command here
            }
        }
    }
}
```

* `agent any` â†’ runs on any Jenkins server/node.
