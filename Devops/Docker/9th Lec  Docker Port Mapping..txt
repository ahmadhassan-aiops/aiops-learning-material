-d --> Detach mode
-a is called attach -->

# docker run --name web-NEW nginx
Above command will not create any binding between host and container port

For mapping , We need to use below command
docker run --name web-NEW -p 9000:80 nginx

-p flag for port mapping

docker run -it -p 8080:80 --name web2 nginx --> Running ? No (Conflict name)
docker run -it -p 8080:80 --name web3 nginx --> running ? YES
docker run -it -p 8001:80 --name web4 nginx --> Running ? YES
